# Provider Network

## Overview

Owns provider credentialing, contracting, network management, and directory data. Responsible for maintaining an adequate network of qualified providers and ensuring accurate provider information for claims processing.

## Responsibilities

- Credential and re-credential providers per NCQA standards
- Negotiate and manage provider contracts and fee schedules
- Monitor and maintain network adequacy across service areas
- Maintain accurate provider demographics and location data
- Manage provider onboarding and termination processes
- Resolve provider data quality issues

## Systems Owned

- **Provider Directory** — master provider data, network status, credentials, and contract fee schedules
