# Provider Relations Manager

## Role Description

Manages provider contracts, negotiates fee schedules, resolves provider payment disputes, and maintains network adequacy. Serves as the primary point of contact between the health plan and contracted providers.

## Key Activities

- Negotiate provider contracts and fee schedules
- Resolve provider payment disputes and claim issues
- Credential new providers and manage re-credentialing cycles
- Monitor network adequacy across service areas and specialties
- Manage provider communications and relationship maintenance
- Onboard new provider groups and practices

## Pain Points

- Provider data quality issues causing claims processing errors
- Complex multi-location practices with varying network status per location
- Fee schedule negotiations requiring analysis of competitive market rates
- Slow credentialing turnaround impacting network adequacy goals
