# Prior Auth

## Definition

Informal abbreviation for pre-authorization. Used interchangeably. Some teams/vendors prefer "prior auth," others "pre-authorization."

## Context

The canonical term in this knowledge base is pre-authorization. When encountering "prior auth" in vendor docs, system logs, or conversations, it refers to the same process. See [Pre-Authorization](pre-authorization.md) for full details.
