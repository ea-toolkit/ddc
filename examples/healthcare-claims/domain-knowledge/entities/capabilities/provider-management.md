# Provider Management

## Overview

Provider credentialing, contracting, network management, and directory data maintenance. Ensures the plan has an adequate network of qualified providers and that their data is accurate for claims processing.

## Sub-Capabilities

- **Provider Credentialing** — Verifying provider qualifications, licenses, and certifications
- **Provider Contracting** — Negotiating and managing provider fee schedules and contract terms
- **Network Management** — Maintaining network adequacy and managing provider participation
- **Provider Data Management** — Maintaining accurate provider demographics, locations, and specialties

## Key Systems

- Provider Directory — master provider data, network status, and fee schedules
