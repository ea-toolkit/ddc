# RetailCo — DDC Knowledge Base

> A synthetic example domain representing a large-scale e-commerce and retail operation.

## About

This is the primary domain knowledge base for the DDC methodology research project. It captures the enterprise architecture of "RetailCo" — a synthetic example domain representing a large-scale e-commerce and retail operation — through demand-driven context curation.

## How It Works

1. A domain problem is posed to an AI agent
2. The agent searches this knowledge base and identifies what's missing
3. The human expert provides the missing domain knowledge
4. The agent curates it into structured entity files
5. The agent re-reads and answers the question

Over hundreds of cycles, the knowledge base converges — new entities decrease, reuse increases, and agent answers improve.

## Domain Areas

- **Store Operations** — physical retail, checkout, inventory
- **E-Commerce** — online shopping, fulfillment, digital experience
- **Supply Chain** — logistics, warehousing, supplier management
- **Range & Product** — product lifecycle, assortment, pricing
- **Customer Engagement** — loyalty, marketing, customer service
- **Corporate Functions** — finance, HR, legal, compliance

## Getting Started

Open Claude Code in this directory and run `/ddc-cycle` to start a cycle.
